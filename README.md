# 📦 Modelagem Conceitual de Banco de Dados — E-commerce  
Desafio do curso DIO: Refinando um Projeto Conceitual de Banco de Dados – E-commerce.

## 🎯 Objetivo do Desafio
Refinar o modelo conceitual proposto no curso, incluindo:

- **Cliente PF ou PJ** – A conta deve ser Pessoa Física *ou* Pessoa Jurídica (nunca as duas).
- **Pagamento múltiplo** – O cliente pode cadastrar mais de uma forma de pagamento.
- **Entrega** – Cada pedido deve possuir *status* e *código de rastreio*.

Este repositório contém o DER atualizado conforme solicitado.
